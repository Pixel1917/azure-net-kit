# Интернационализация (i18n)

Система поддержки множественных языков и локализации приложения.

## 🌍 Основы i18n

**Файл:** `src/lib/i18n/index.ts`

Модуль интернационализации основан на библиотеке `edges-svelte-translations` и предоставляет удобные инструменты для многоязычных приложений.

### Основное использование

```typescript
import {} from /* i18n утилиты */ '@azure-net/kit/i18n';
```

### Инициализация i18n

```typescript
// src/lib/i18n/config.ts
import { createI18n } from 'edges-svelte-translations';
import { en } from './translations/en';
import { ru } from './translations/ru';

export const i18n = createI18n({
	defaultLocale: 'ru',
	locales: {
		en,
		ru
	},

	// Функция для получения локали из запроса
	detectLocale: (event) => {
		// Приоритет: параметр URL > cookie > заголовок Accept-Language
		const urlLocale = event.url.searchParams.get('lang');
		if (urlLocale && ['en', 'ru'].includes(urlLocale)) {
			return urlLocale;
		}

		const cookieLocale = event.cookies.get('locale');
		if (cookieLocale && ['en', 'ru'].includes(cookieLocale)) {
			return cookieLocale;
		}

		const acceptLanguage = event.request.headers.get('accept-language');
		if (acceptLanguage) {
			if (acceptLanguage.includes('ru')) return 'ru';
			if (acceptLanguage.includes('en')) return 'en';
		}

		return 'ru';
	}
});

// Экспорт хелперов
export const { t, locale, setLocale, availableLocales } = i18n;
```

## 🎯 Использование в компонентах

### Базовое использование

```svelte
<!-- src/routes/+layout.svelte -->
<script>
	import { t, locale, setLocale, availableLocales } from '$lib/i18n/config';
	import { page } from '$app/stores';

	// Установка локали из данных страницы
	$: if ($page.data.locale) {
		setLocale($page.data.locale);
	}

	const changeLanguage = (newLocale) => {
		setLocale(newLocale);
		// Сохранение в cookie
		document.cookie = `locale=${newLocale}; path=/; max-age=${60 * 60 * 24 * 365}`;
		// Перезагрузка страницы для применения изменений на сервере
		window.location.reload();
	};
</script>

<header>
	<nav>
		<a href="/">{$t('navigation.home')}</a>
		<a href="/about">{$t('navigation.about')}</a>
		<a href="/contact">{$t('navigation.contact')}</a>
	</nav>

	<!-- Переключатель языка -->
	<div class="language-switcher">
		{#each availableLocales as lang}
			<button onclick={() => changeLanguage(lang)} class:active={$locale === lang}>
				{lang.toUpperCase()}
			</button>
		{/each}
	</div>
</header>

<main>
	<slot />
</main>
```

## 📄 Серверная интеграция

### Load функции с переводами

```typescript
// src/routes/+layout.server.ts
import { i18n } from '$lib/i18n/config';
import type { LayoutServerLoad } from './$types';

export const load: LayoutServerLoad = async ({ event }) => {
	// Определение локали
	const locale = i18n.detectLocale(event);

	// Установка локали для серверного контекста
	i18n.setLocale(locale);

	return {
		locale,
		translations: i18n.getTranslations(locale)
	};
};
```

### Server Actions с переводами

```typescript
// src/routes/users/+page.server.ts
import { createServerAction } from '@azure-net/kit';
import { i18n } from '$lib/i18n/config';
import { UserSchema } from '$lib/schemas';

export const actions = {
	create: createServerAction(async ({ context, utils }) => {
		const { fail, redirect } = utils;

		// Установка локали из cookie или заголовков
		const locale = i18n.detectLocale({
			url: context.url,
			cookies: context.cookies,
			request: context.request
		});
		i18n.setLocale(locale);

		const data = await context.request.formData();

		try {
			const userData = UserSchema.from(data).json();
			await createUser(userData);

			return {
				success: true,
				message: i18n.t('users.userCreated')
			};
		} catch (error) {
			const schemaErrors = UserSchema.getSchemaError(error);
			if (schemaErrors) {
				return fail(422, {
					errors: schemaErrors,
					message: i18n.t('validation.error')
				});
			}

			return fail(500, {
				message: i18n.t('common.error')
			});
		}
	})
};
```

## 🔧 Интеграция с валидацией

### Локализованные правила валидации

```typescript
// src/lib/schemas/validation.ts
import { createSchemaFactory } from '@azure-net/kit';
import { t } from '$lib/i18n/config';

export const createLocalizedRules = () => ({
	required:
		() =>
		({ val }) =>
			!val ? t('validation.required') : undefined,

	email:
		() =>
		({ val }) => {
			if (!val) return undefined;
			const emailRegex = /\S+@\S+\.\S+/;
			return !emailRegex.test(val) ? t('validation.email') : undefined;
		},

	minLength:
		(min: number) =>
		({ val }) => {
			if (!val) return undefined;
			return val.length < min ? t('validation.minLength', { min }) : undefined;
		},

	maxLength:
		(max: number) =>
		({ val }) => {
			if (!val) return undefined;
			return val.length > max ? t('validation.maxLength', { max }) : undefined;
		},

	confirmPassword:
		(originalField: string) =>
		({ val, listValues }) => {
			const original = listValues?.[originalField];
			return val !== original ? t('validation.passwordMismatch') : undefined;
		}
});

// Использование в схемах
const createUserSchema = createSchemaFactory(createLocalizedRules());

export const UserSchema = createUserSchema()
	.rules((r) => ({
		name: [r.required(), r.minLength(2), r.maxLength(50)],
		email: [r.required(), r.email()],
		password: [r.required(), r.minLength(8)],
		confirmPassword: [r.required(), r.confirmPassword('password')]
	}))
	.create();
```
