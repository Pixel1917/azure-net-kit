# Документация Azure-Net/Kit для Svelte 5

Azure-Net/Kit - это мощный инструментарий для разработки веб-приложений на Svelte 5 с использованием архитектуры Domain-Driven Design (DDD).

## 🚀 Быстрый старт

### Установка

```bash
npm install @azure-net/kit
```

### Основное использование

```typescript
import { createBoundaryProvider, createServerAction } from '@azure-net/kit';
```

## 📖 Основные экспорты пакета

Пакет предоставляет следующие основные экспорты:

- `@azure-net/kit` - Основные функции из core
- `@azure-net/kit/constants` - Константы (маски и т.д.)
- `@azure-net/kit/infra` - Инфраструктурные компоненты
- `@azure-net/kit/schema` - Правила валидации
- `@azure-net/kit/edges` - Края системы
- `@azure-net/kit/edges/server` - Серверная интеграция
- `@azure-net/kit/edges/state` - Управление состоянием
- `@azure-net/kit/edges/context` - Управление контекстом
- `@azure-net/kit/tools` - Дополнительные инструменты
- `@azure-net/kit/i18n` - Интернационализация

## 🏗️ Архитектурные принципы

Пакет следует принципам Domain-Driven Design (DDD) и чистой архитектуры:

1. **Слой домена** - содержит бизнес-логику и правила
2. **Слой приложения** - координирует взаимодействие между слоями
3. **Слой инфраструктуры** - обеспечивает техническую реализацию

## 📝 Примеры использования

Все примеры использования можно найти в папке `src/app` данного проекта, где демонстрируется полноценное приложение с использованием всех возможностей пакета.
